# Temperature-Based LED Control System

This project implements a dual-mode embedded system using Arduino. It reads temperature from a TMP36 sensor and controls an LED based on real-time conditions.

The system supports both automatic operation (sensor-based) and manual control using serial communication.



## Features

. Dual mode operation (Auto + Manual)
. Temperature-based LED control
. Manual control using Serial Monitor
. Real-time monitoring and alert system



## How to Run

1. Open the project in Tinkercad or Arduino IDE
2. Connect TMP36 sensor to A0
3. Open Serial Monitor
4. Type:

    `A` → Auto mode
    `M` → Manual mode
    `1` → LED ON
    `0` → LED OFF
