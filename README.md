
# Temperature-Based LED Control System

This is a simple Arduino project where I built a system that reads temperature using a TMP36 sensor and controls an LED based on that value.

If the temperature goes above 40°C, the LED turns ON. Otherwise, it stays OFF.

# What I learned

. How to connect sensors and LEDs using Arduino
. Converting analog sensor data into temperature
. Writing simple logic to control outputs

# Components Used

. Arduino UNO
. TMP36 Temperature Sensor
. LED
. Resistor

# Tools

. Tinkercad
. Arduino (C programming)
